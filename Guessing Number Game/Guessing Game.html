Number Guessing Game in Console Using JavaScript

PROPERTIES

1- User will Enter the Maximum Range of Number

2- Then User will Guess the Number Having Various Attempts

3- If User want to Exit from Game Then he can by entering "quit"

4- Hint will be Given so that user came to know whether guess is Larger or Smaller than Entered Number

5- Once You Guessed the Number correctly . Game will be Finished Having Congrats Prompt on Screen
